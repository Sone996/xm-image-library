# XM Image Library

Small Angular application for browsing images, opening image details, and managing favorites.

## Requirements

- Node.js 20 or newer (recommended 22)
- npm 10 or newer

## Setup

Install project dependencies:

```bash
npm install
```

## Start the Project

Run the development server:

```bash
npm start
```

The app will be available at:

```text
http://localhost:4200
```

## Run Tests

Run the full Jest test suite once:

```bash
npm test
```

Run tests in watch mode:

```bash
npm run test:watch
```

## Other Useful Commands

Build the project:

```bash
npm run build
```

Run linting:

```bash
npm run lint
```

Format the codebase:

```bash
npm run format
```
